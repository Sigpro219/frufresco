description: Guarda el progreso actual en la rama CORE (Motor del sistema)
---

Este comando asegura que tus cambios locales estén respaldados en GitHub bajo la identidad de FruFresco.

// turbo-all

1. Agregar todos los cambios: `git add .`
2. Crear un punto de restauración: `git commit -m "Actualización de desarrollo en CORE"`
3. Subir a la nube: `git push origin core`
